# AI Telegram Assistant для Центр Красок №1

## Описание проекта

AI Telegram Assistant — это Telegram-бот с AI-консультантом для компании **Центр Красок №1**.

Бот отвечает на вопросы пользователей о компании, товарах, услугах, брендах, контактах, популярных продуктах и сотрудничестве.

Пользователь может писать обычные сообщения без команд и меню. Бот работает в формате AI-чата.

## Цель проекта

Цель проекта — создать рабочий MVP Telegram-бота, который:

* отвечает на вопросы о компании;
* использует собранную базу знаний;
* помогает клиентам получать информацию о товарах и услугах;
* ограничивает ответы только информацией о компании;
* демонстрирует интеграцию Telegram Bot API и AI API.

## Возможности бота

Бот может отвечать на вопросы:

* чем занимается компания;
* какие услуги предоставляет Центр Красок №1;
* какие товары есть в ассортименте;
* какие бренды представлены;
* какие товары популярные;
* где находятся магазины;
* есть ли доставка;
* есть ли колеровка;
* работает ли компания с дизайнерами и партнерами;
* что означают термины из сферы лакокрасочных материалов.

## Описание файлов

### bot.py

Главный файл Telegram-бота.

Он:

* запускает Telegram-бота;
* принимает сообщения пользователей;
* обрабатывает команду `/start`;
* отвечает на приветствие, благодарность и прощание;
* отправляет вопросы пользователя в AI-сервис;
* хранит краткую историю диалога.

### ai_service.py

Файл для работы с AI API.

Он:

* подключается к OpenRouter API;
* загружает базу знаний компании;
* формирует системный prompt;
* отправляет вопрос пользователя в AI-модель;
* возвращает ответ Telegram-боту.

### knowledge_base.py

Файл для загрузки базы знаний.

Он:

* открывает папку `data`;
* читает все `.txt` файлы;
* объединяет информацию в один текст;
* передает эту информацию AI-модели.

### data/

Папка с базой знаний компании.

В ней хранятся структурированные текстовые файлы с информацией о компании, товарах, услугах, контактах, брендах и FAQ.

## Как работает бот

1. Пользователь отправляет сообщение в Telegram.
2. `bot.py` получает сообщение.
3. Сообщение передается в функцию `ask_ai()`.
4. `ai_service.py` загружает базу знаний через `knowledge_base.py`.
5. AI получает вопрос пользователя и информацию из файлов `data`.
6. AI формирует ответ только на основе базы знаний.
7. Бот отправляет ответ пользователю в Telegram.

## Установка проекта

### 1. Клонировать или скачать проект

```bash
git clone <https://github.com/aizatkenges/centr-krasok-ai-assistant.git>
cd <kraski_ai_bot>
```

### 2. Установить зависимости

```bash
pip install aiogram python-dotenv openai
```

### 3. Запустить бота

```bash
python bot.py
```

После запуска бот начнет принимать сообщения в Telegram.

## Ограничение галлюцинаций

Чтобы AI не придумывал информацию, в системном prompt указаны правила:

* отвечать только на основе базы знаний;
* не придумывать факты;
* не отвечать на вопросы, не связанные с компанией;
* если информации нет, писать:
  “В моей базе нет точной информации по этому вопросу.”

## Контекст диалога

В проекте используется `user_history`.

Он хранит последние сообщения пользователя и ответы бота. Это помогает сохранять контекст диалога и делать ответы более связанными.
